# Monorepo starter

A monorepo supports front-end and back-end separation, synchronizes types based on contracts
and supports docker build for the corresponding app's service.

## Tech stack

**common**

- eslint
- typescript
- orpc
- vite
- turborepo

**ui**

- react
- react-query
- tailwindcss
